## 常用符号

| 写法                       | 效果              |
| -------------------------- | ----------------- |
| \quad or \                 | 空格              |
| \frac                      | 分式              |
| \bar x                     | x拔               |
| \pm                        | 正负              |
| \sqrt[2]{x}                | 二次根号下x       |
| \infty                     | 无穷大            |
| \le                        | <=                |
| \ge                        | >=                |
| \times                     | 乘以              |
| \approx                    | ≈                 |
| \prime                     | 八的左撇          |
| \ll                        | <<                |
| \gg                        | >>                |
| \cdots                     | ...               |
| \vec{a}                    | a向量（箭头）     |
| \int{x}dx                  | x的不定积分       |
| \int_{1}^{2}{x}dx          | x的从1到2的定积分 |
| \lim{a+b}                  | 极限              |
| \lim_{n\rightarrow+\infty} | 极限              |
| \sum{a}                    | 求和              |
| \sum_{n=1}^{100}{a_n}      | 求和              |
| \prod{x}                   | 连乘              |
| \prod_{n=1}^{99}{x_n}      | 连乘              |
